## How to Run the Code

### Prerequisites
- Java JDK 8 or above installed
- Any Java IDE or terminal

### Steps to Run

1. Clone the repository:
```bash
git clone https://github.com/abxijth/oop-term-project-student-attendance-management-system.git
```
2. Navigate inside Phase-3/Program/ directory

3. Open the project in your preferred IDE.
    Compile the Java files:
      javac *.java
3. Run the main file:
    java AttendanceManagementSystem
